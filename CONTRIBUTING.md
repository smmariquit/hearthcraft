# Contributing to HearthCraft

Thanks for helping improve this site.

## Report issues

Open a [GitHub issue](https://github.com/smmariquit/hearthcraft/issues) for bugs, broken media, or content corrections.

## Developers

### Setup

```sh
npm install
npm run dev       # Next.js — see package.json for current scripts
```

### Pull requests

1. Branch from **`main`**.
2. Use [Conventional Commits](https://www.conventionalcommits.org/) (`fix: …`, `feat: …`).
3. Run `npm run build` before requesting review for substantive changes.
4. Do not commit secrets or large binary assets unless they belong in `public/`.
